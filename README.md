# WeekPrep

**Your weekly meal planning, simplified**

Take the stress out of weeknight dinners. WeekPrep helps busy professionals plan, organize, and shop for the week ahead in minutes. Discover recipes, drag them into your weekly calendar, and automatically generate your shopping list. Less time planning, more time living.

## ✨ Key Features

- 📅 **Visual Weekly Calendar** - Drag and drop recipes into your meal plan
- 🔍 **Smart Recipe Search** - Discover recipes by name, ingredient, or cuisine
- 🛒 **Auto-Generated Shopping Lists** - Automatically creates shopping lists from your meal plan
- ⏱️ **Quick Planning** - Plan your entire week in under 10 minutes
- 🎨 **Modern UI/UX** - Clean, professional design inspired by FANG companies
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile

## 🚀 Tech Stack

This project was built using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.0 with modern Angular features including signals, standalone components, and the Angular CDK for drag-and-drop functionality.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
